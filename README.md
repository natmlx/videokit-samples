# VideoKit Samples
This project is a collection of samples showing how to build interactive experiences using Unity Engine and VideoKit.

## Requirements
- Unity 2022.3+

## Supported Platforms
- Android API level 24+
- iOS 14+
- macOS 11+ (Apple Silicon and macOS)
- Windows 10+ (64-bit only)
- WebGL:
    - Chrome 91+
    - Firefox 90+
    - Safari 16.4+
